# 🌦️ ClimaHora

ClimaHora es una Progressive Web App (PWA) moderna enfocada en mostrar información meteorológica en tiempo real de manera rápida, ligera y visualmente atractiva.

🔗 Demo en vivo:  
https://climahoradelmundo.netlify.app/

---

# ✨ Características

- 🌍 Consulta del clima en diferentes ciudades del mundo
- 🌡️ Temperatura actual y sensación térmica
- 💧 Humedad y velocidad del viento
- ☁️ Estado del clima en tiempo real
- 🕒 Pronóstico por horas
- 📱 Diseño responsive para móviles y PC
- ⚡ Carga rápida y optimizada
- 📲 Instalable como aplicación (PWA)
- 🔄 Funcionamiento offline básico mediante Service Worker

---

# 🚀 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Service Workers
- Manifest.json
- API meteorológica
- Netlify

# 📥 Instalación local

## 1. Clonar el repositorio

```bash
git clone https://github.com/JuanReyesVazquez23/ClimaHora.git
```

## 2. Entrar al proyecto

```bash
cd ClimaHora
```

## 3. Abrir el proyecto

Abrir `index.html` en el navegador.
